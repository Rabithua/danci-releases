# Danci Releases

[Danci](https://github.com/Rabithua/Danci) macOS 版本的分发仓库。

## 下载

前往 [Releases](https://github.com/Rabithua/danci-releases/releases) 下载最新版本的 DanciMac。

## 自动更新

DanciMac 内置了 [Sparkle](https://sparkle-project.org) 自动更新框架。应用会自动检查更新，也可以在设置 > 关于 > 检查更新中手动触发。

更新源：[appcast.xml](https://rabithua.github.io/danci-releases/appcast.xml)

## 关于 Danci

Danci 是一款中英词典应用，支持 iOS 和 macOS。
